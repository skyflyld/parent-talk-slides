# 家长会分享 · 交互式课件

> 考后复盘 · 周末手机管理 · 有效陪伴

一个基于三张家庭教育理念幻灯片的交互式课件，适用于家长会现场分享（约15分钟）。

## 使用方式

1. 打开 `index.html` 即可全屏演示
2. 键盘 ← → / 触摸滑动切换页面
3. 点击图片可放大查看
4. 交互式元素（折叠面板、勾选清单、切换卡片）可在讲解时与家长互动

## 三张图片素材

| 图片 | 主题 | 用途 |
|------|------|------|
| slide-low-voice.jpg | 低声教育 | 考后复盘段的科学依据 |
| slide-boundary-metaphor.jpg | 悬崖与水的边界 | 手机管理和沟通的贯穿隐喻 |
| slide-change-cognition.jpg | 改变认知 | 陪伴段的责任定位论据 |

## 课件结构

1. **开场** — 今晚聊三件事（折叠展开）
2. **低声教育** — 吼是最没用的教育（图1）
3. **考后复盘三步法** — 先感受→找亮点→他方案
4. **边界隐喻** — 悬崖还是水（图2，贯穿隐喻）
5. **手机管理** — 定公约·放手机·填时间（勾选互动）
6. **改变认知** — 两个最常见误区（图3）
7. **有效陪伴** — 在陪他还是陪作业？（切换卡片互动）
8. **总结** — 三张图·三句话

## 现场使用建议

- 可以投屏放映，配合口头讲解
- 交互元素可在讲到对应段落时"请各位家长点一下"
- 三张图建议在对应段落停留在屏幕上5-10秒再翻页
